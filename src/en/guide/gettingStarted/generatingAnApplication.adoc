To get started quickly with Grails it is often useful to use a feature called <<scaffolding,Scaffolding>> to generate the skeleton of an application. To do this use one of the `generate-*` commands such as link:../ref/Command%20Line/generate-all.html[generate-all], which will generate a link:theWebLayer.html#controllers[controller] (and its unit test) and the associated <<gsp,views>>:

[source,groovy]
----
grails generate-all helloworld.Book
----
