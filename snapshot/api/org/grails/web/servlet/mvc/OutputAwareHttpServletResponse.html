[Groovy] Class OutputAwareHttpServletResponse

        • org.grails.web.servlet.mvc.OutputAwareHttpServletResponse 

        • @groovy.transform.CompileStatic
@groovy.transform.InheritConstructors
class OutputAwareHttpServletResponse
extends javax.servlet.http.HttpServletResponseWrapper

          Tracks whether getOutputStream() was called in order to prevent calls to getWriter() after getOutputStream() has been called
