Package: org.grails.io.support

[Java] Class AntPathMatcher.AntPathStringMatcher

    • org.grails.io.support.AntPathMatcher.AntPathStringMatcher 

    • static class AntPathMatcher.AntPathStringMatcher

      Package-protected helper class for org.springframework.util.AntPathMatcher. Tests whether or not a string matches against a pattern via a java.util.regex.Pattern.

      The pattern may contain special characters: '*' means zero or more characters; '?' means one and only one character; '{' and '}' indicate a URI template pattern. For example /users/{user}.
