[Groovy] Class TypeConvertingMap

        • grails.util.TypeConvertingMap 

        • @groovy.transform.CompileStatic
class TypeConvertingMap
extends AbstractTypeConvertingMap

          TypeConvertingMap is a Map with type conversion capabilities. Type converting maps have no inherent ordering. Two maps with identical entries but arranged in a different order internally are considered equal.
