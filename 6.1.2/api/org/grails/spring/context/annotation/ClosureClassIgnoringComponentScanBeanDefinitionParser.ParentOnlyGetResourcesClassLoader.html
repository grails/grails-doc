[Java] Class ClosureClassIgnoringComponentScanBeanDefinitionParser.ParentOnlyGetResourcesClassLoader

      • org.grails.spring.context.annotation.ClosureClassIgnoringComponentScanBeanDefinitionParser.ParentOnlyGetResourcesClassLoader 

      • private static final class ClosureClassIgnoringComponentScanBeanDefinitionParser.ParentOnlyGetResourcesClassLoader
extends java.lang.ClassLoader

        This ClassLoader is used to restrict getResources & getResource methods only to the parent ClassLoader. getResources/getResource usually search all parent level classloaders. (look at details in source code of java.lang.ClassLoader.getResources)
